Instagram Login Page Clone

A simple and responsive Instagram Login Page UI clone built using HTML and CSS. This project is ideal for beginners to understand layout design, styling, and basic front-end structure.

Features
Clean and modern UI inspired by Instagram
Fully responsive design
Styled login form with input fields
Social media login section
App download section (Google Play & App Store)
Footer with links

Technologies Used
HTML5 – Structure of the webpage
CSS3 – Styling and layout design

Project Structure

│── index.html       # Main HTML file
│── styles.css       # Styling file
